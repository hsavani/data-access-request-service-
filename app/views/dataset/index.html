Data set(s)

Important: Ensuring the information you enter is saved

Details you enter for each data set will not be saved until you have completed the information in full. Once information on a single data set has been saved, the data set will show on the ‘You have added the below data set(s)’ page within the data set section of the service.

In this section, you are required to select Secure Data Environment (SDE) or data dissemination. Secure Data Environment has replaced the Trusted Research Environment (TRE).

You should note that the total estimated cost does not include data received via SDE.

If you select to receive NHS Digital’s data sets in SDE, your Case Officer will contact you.

Data set(s) selection

This data set(s) selection part of your Data Sharing Agreement (DSA) application enables you to:

  • choose and select data set(s)
  • choose year-to-date, historic annual or future release data
  • choose and select fields within a data set(s)
  • choose and select a data set(s) frequency
  • provide information that helps NHS Digital link data to a group of Participants you are going to provide
  • understand if your selected data set(s) will be provided by NHS Digital in 'SDE' or 'data dissemination’ format

Before you start

Before you start, we recommend that this section is completed (or supported) by someone who will be using the data.

Make sure you understand the avilable data sets, fields and services using our:

  • data sets catalogue (opens in a new window)
  • metadata dashboard (opens in a new window)
  • metadata store (opens in a new window)
  • data dissemination service (opens in a new window)

Understanding data sets and the services that you can request through DARS (opens in a new window) will help you to select data you need that supports both the purpose and legal basis or bases.

Understanding charges (opens in a new window) will help you to breakdown the costs associated with requesting data.

Services

The most common ways to receive data from NHS Digital are through data dissemination and SDE.

The DARS team deal with a multitude of data dissemination (opens in a new window) requests that include requests for data sets, linked data sets, tabulations, bespoke linkage and patient tracking.

NHS Digital’s Secure Data Environment (SDE) service (opens in a new window) provides approved researchers and analysts with access to essential, de-identified health data from national health settings, to deliver quick answers to vital health-related questions.

If you need help with your data set(s) selection, contact NHS Digital on 0300 303 5678, or email data.applications@nhsdigital.nhs.uk
